02. 이벤트 핸들러 프로퍼티 방식

        window 객체와 Document, HTMLElement 타입의 DOM 노드 객체는
        이벤트에 대응하는 이벤트 핸들러 프로퍼티를 가지고 있다.
        이벤트 핸들러 프로퍼티 키는 이벤트 핸들러 어트리뷰트와 동일(on 접두사 + 이벤트타입)
        이벤트 핸들러 프로퍼티에 함수를 바잉ㄴ딩하면 이벤트 핸들러가 등록된다.

        어트리뷰트 단점 : HTML, JS 섞이는 문제가 발생
        프로퍼티 방식 -> 어트리뷰트의 단점을 해결할 수 이ㄸ다.
        프로퍼티 단점 : 이벤트 핸들러 프로퍼티에 1개의 이벤트만 넣을 수 이다.
