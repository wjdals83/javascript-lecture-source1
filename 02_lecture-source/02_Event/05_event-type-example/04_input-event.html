04. 사용자 입력 양식 이벤트







0/5000자

폼 요소 값 다루기

        input, textarea : input.value 또는 input.checked(checkbox 또는 radio)
        select.options : option 하위 요소들을 담고 있는 컬렉션
        select.value : 현재 선택 된 option 값
        select.selectedIndex : 현재 선택 된 option의 번호(인덱스 체계)

focus, blur 이벤트

        focus : 사용자가 폼 요소를 클릭하거나 tab 키를 눌러 요소로 이동했을 때 발생하는 이벤트이다.
        blur : 사용자가 다른 곳으라 클릭하거나 tab 키를 눌러 다음 폼 필드로 이동했을 떄 발생하는 이벤트이다.
        또한 focus, blur 메소드로 요소에 포거스를 주거나 제거할 수 있다.

focusin, focusout

        focus 이벤트는 해당 입력 필드에서만 동작하고, 버블링 되지 않는다.
        버블링이 필요하다면 focusin, focusout 이벤트를 사용한다.

change 이벤트

        change 이벤트는 요소 변경이 끝나면 발생하는 이벤트이다.
        select/checkbox/radio 경우 선택 값이 변경 된 직후 이벤트가 발생하지만
        텍스트 요소인 경우 변경 후 포커스를 잃었을 때 이벤트가 발생한다.
