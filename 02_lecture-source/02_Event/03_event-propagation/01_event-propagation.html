01. 이벤트 전파

        생성된 이벤트 객체는 이벤트를 발생시킨 DOM 요소인 타겟
        (event target)을 중심으로 DOM 트리를 통해 전파된다.
        1. 캡쳐링 단계(capturing phase) : 이벤트가 상위 요소에서 하위 요소 방향으로 전파
        2. 타겟 단계(target phase) : 이벤트가 이벤트 타겟에 도달
        3. 버블링 단계(bubbling phase) : 이벤트가 하위 요소에서 상위 요소로 전파

        전파의 특성을 파악하면, 이벤트를 위임하거나, 중지, 단계별 처리를 할 수 있다.
