01. 노드 정보 취득

        Node.prototype.nodeType : 노드 객체의 종류, 즉 노드 타입을 나타내는 상수 반환
            element : 요소 타입을 나타내면 상수 1
            text : 텍스트 노드 타입을 나타내는 상수 3
            document : 문서 노드 타입을 나타내는 상수 9
        Node.prototype.nodeName : 노드의 이름을 문자열로 반환
            element : 문자열로 태그 이름 반환 -> 대문자
            text : 문자열 "#text" 반환
            document : 문자열 "#document" 반환
