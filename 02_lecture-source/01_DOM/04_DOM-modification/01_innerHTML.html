01. innerHTML

        Element.prototype.innerHTML 프로퍼티는 getter, setter 모두 존재하는 프로퍼티다.
        요소 노드의 html 마크업을 취득하거나 변경한다. 
        textContent 프로퍼티는 html 마크업을 무시하고 텍스트만 반환하지만,
        innerHTML 프로퍼티는 HTML 마크업이 포함 된 문자열을 그대로 반환한다.
태그 엘리먼트의 값을 읽거나, 변경할 때 innerHTML속성을 사용한다.
        innerHTML 프로퍼티를 사용하면 DOM 조작이 간편하고 직관적이라는 장버이 있다.
        사용자한테 입력 받은 데이터를 그대로 innerHTML 에 할당하는 것으내 보안 취약.
        악성 코드를 html 마크업 내에 포함한다면 파싱(번역) 과정에서 그대로 실해오딜 수 있다.

        innerHTML 포로퍼티에 html 마크업 문자열을 할당하는 경우 욧 노드의
        모든 자식을 제거하고 새롭게 ㅏ탄생한 Html 마크업 문자열을 파싱하여
        변경한다는 단점이 잇다.

        새로운 요소를 삽입할 때, 삽일 될 위치를 지정할 수 없다.
